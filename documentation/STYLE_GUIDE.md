# Mella Boutique Style Guide

This document outlines the official style guide for the Mella Boutique website, including color palettes, typography, and component styling.

## Color Theme

Our color theme is designed to be elegant, modern, and luxurious.

### Primary & Accent

-   **Primary/Accent:** `#FFCB74` (The main gold color for buttons, highlights, and calls to action.)
-   **Primary/Accent Content:** `#111111` (The black text used on gold buttons.)

### Dark Theme

-   **Base:** `#2F2F2F` (The primary background color for dark mode.)
-   **Base Content:** `#F6F6F6` (The light text color used on dark backgrounds.)

### Light Theme

-   **Base:** `#F6F6F6` (The primary background color for light mode.)
-   **Base Content:** `#111111` (The black text color used on light backgrounds.)

## Typography

Our typography is designed to be clean, readable, and elegant. We use two primary fonts:

-   **Cormorant:** A serif font for headings and display text.
-   **Inter:** A sans-serif font for body text and UI elements.
